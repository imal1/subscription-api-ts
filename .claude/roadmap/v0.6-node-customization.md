# v0.6 — 节点定制 & 规则引擎

**状态：待开发**

## 版本目标

让用户能够自定义 Clash 配置的输出内容：代理组策略、分流规则、DNS 配置等，而不仅仅是默认的通用模板。

## 核心功能

### 1. Clash 配置模板系统

- **内置模板**：提供 3-5 个预设模板
  - 默认（全局代理）
  - 大陆白名单（国内直连、国外代理）
  - 游戏优化（UDP 优先）
  - 流媒体解锁（按平台分流）
  - 极简（最小配置，仅代理节点）
- **模板预览**：选择模板后实时预览生成的 Clash 配置结构
- **模板变量**：支持 `{{proxyNames}}`、`{{nodeCount}}` 等变量替换

### 2. 分流规则管理

新增 `/rules` 路由：

- **规则集管理**：预置常见规则集（广告屏蔽、隐私保护、国内域名）
- **自定义规则**：支持 Domain / Domain-Suffix / Domain-Keyword / IP-CIDR / GeoIP / MATCH
- **规则排序**：拖拽调整规则优先级
- **规则测试**：输入域名测试匹配结果

### 3. 代理组配置

- **自定义代理组**：创建、编辑、删除代理组
- **组类型**：select / url-test / fallback / load-balance
- **组内节点**：手动选择节点或使用过滤器（如"所有 vless 节点"）
- **测速 URL**：自定义 url-test / fallback 的测速地址
- **间隔配置**：url-test / fallback 的检测间隔

### 4. DNS 配置

- **DNS 服务器**：自定义上游 DNS 列表
- **DNS 策略**：fake-ip / redir-host 模式切换
- **DNS 规则**：指定域名使用特定 DNS 服务器

### 5. 配置导出选项

- **YAML**：标准 Clash 配置格式
- **URI**：Clash 订阅链接格式（`clash://install-config?url=...`）
- **二维码**：生成 Clash 配置导入二维码

## 验收标准

- [ ] 5 个预设模板可选择，切换后实时预览
- [ ] 自定义分流规则可增删改排序
- [ ] 自定义代理组功能完整
- [ ] DNS 配置可编辑并生效
- [ ] 生成的 Clash 配置通过 mihomo 验证
- [ ] 规则测试功能正确返回匹配结果
- [ ] 配置导出 3 种格式均可用
