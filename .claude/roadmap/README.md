v0.1 ──→ v0.2 ──→ v1.0 ──→ v1.1 ──→ v1.2 ──→ v1.3 ──→ v1.4 ──→ v1.5 ──→ v1.6 ──→ v1.7 ──→ v1.8

**总计预计工作量**：45-59 天（单人开发）

## 版本关系

每个版本依赖前一个版本的功能，按顺序推进。v0.1 是基线，v1.0 是多节点架构的重大升级，v1.8 是生产就绪的里程碑。
